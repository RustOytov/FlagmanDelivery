# FlagmanDelivery

FlagmanDelivery — это full-stack проект сервиса доставки, который включает iOS-приложение и backend API

## Что есть в проекте

Проект состоит из двух частей:

- FlagmanDelivery/ — iOS-приложение на SwiftUI
- hac.new/ — backend на FastAPI

## Роли в системе

В приложении реализованы 3 роли:

- Customer — регистрация, просмотр каталога, оформление заказа
- Courier — регистрация, выход на смену, принятие заказа, доставка заказа
- Owner — регистрация, создание организации, настройка зон доставки, управление меню и заказами

### iOS

- PhotosUI

### Backend
- Python
- FastAPI
- SQLAlchemy
- Pydantic
- PostgreSQL
- PostGIS
- GeoAlchemy2
- Shapely
- Docker Compose

## Основной функционал

- регистрация и авторизация пользователей
- работа с несколькими ролями
- создание и получение заказов через backend
- отображение доступных заказов для курьера
- управление меню и организациями для owner
- работа с картами и зонами доставки
- подтверждение доставки курьером с загрузкой фотографии

## Безопасность

В проекте уже реализованы базовые механизмы защиты данных:

- пароли на backend хранятся в виде хеша
- access и refresh токены используются для авторизации
- токены в iOS-приложении сохраняются в Keychain
